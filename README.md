Naruto Hand Jutsu (Web Camera Project)

This is a simple web project inspired by the Naruto anime.
The visual effects are inspired by Rasenshuriken and Chidori techniques.

Using MediaPipe hand tracking, the camera detects your hands and triggers anime-style energy effects when you open your hand.

Features

Real-time hand tracking using MediaPipe

Naruto-inspired energy effects

Works with your webcam in the browser

Two different effects for left and right hand

Technologies Used

HTML

CSS

JavaScript

MediaPipe Hands

How to Run

Download or clone the repository.

Open the project folder in VS Code.

Run index.html using Live Server.

Allow camera access.

Open your hand in front of the camera to activate the effects.
